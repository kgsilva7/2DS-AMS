O Diagrama de Casos de Uso da Vixel Sports tem como objetivo descrever, de maneira simples e visual, todas as interações que os usuários podem realizar no sistema, destacando o que cada tipo de usuário faz e quais funcionalidades o sistema deve oferecer. Esse tipo de diagrama ajuda tanto no entendimento dos requisitos do sistema quanto no planejamento do desenvolvimento. O Cliente é o usuário que navega pela loja online para visualizar e adquirir produtos. Para ele, o sistema oferece as seguintes funcionalidades: Visualizar produtos: permite consultar os itens disponíveis na loja. Manter produtos no carrinho: adicionar, remover e atualizar produtos no carrinho de compras. Realizar compra dos produtos: finalizar a compra, utilizando o carrinho já montado. Enviar mensagem para suporte: registrar dúvidas ou problemas durante o uso da plataforma. Manter cliente: engloba operações relacionadas ao seu cadastro (incluso como requisito sempre que realiza ações de compra). Essas funcionalidades estão relacionadas entre si com os relacionamentos «include» e «extend», indicando processos obrigatórios ou opcionais dentro do fluxo principal. Administrador (Admin) O administrador é responsável por manter o funcionamento da loja. No diagrama, ele possui acesso à funcionalidade: Manter suporte: gerenciar as mensagens enviadas pelos clientes, respondendo e acompanhando atendimentos. Em síntese O diagrama demonstra como o sistema deve funcionar do ponto de vista dos usuários, mostrando: Quem utiliza o sistema; Que ações cada usuário pode realizar; Como essas ações se relacionam; Quais funcionalidades são essenciais para o funcionamento da loja virtual.
Ele representa uma visão geral e estruturada dos requisitos funcionais da plataforma de e-commerce da Vixel Sports, auxiliando no desenvolvimento de um sistema organizado, coerente e de fácil entendimento.
